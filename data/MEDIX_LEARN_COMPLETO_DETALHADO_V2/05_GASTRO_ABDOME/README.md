# MEDIX Learn — Gastro/Abdome V3 Curado

Pacote criado no padrão novo do MEDIX Learn:

- linguagem médica direta;
- texto em segunda pessoa (“você”);
- conteúdo denso, mas organizado em blocos curtos;
- fisiopatologia aplicada à semiologia;
- foco em decisão clínica e OSCE;
- mini-casos reais;
- questões ativas específicas;
- mapa final útil.

`content/learn/gastro-abdome/`

mantendo os nomes/rotas que o app já usa.
