# MEDIX Learn Completo — V3 Curado

> Material educacional para treino clínico e OSCE. Não substitui supervisão docente, diretrizes locais ou atendimento médico real.

## O que é este pacote

Este pacote contém a versão **V3 Curada** do **MEDIX Learn**, organizada em sistemas e trilhas clínicas.

A proposta é ensinar você a raciocinar, não apenas revisar conteúdo.

```text
conceito → mecanismo → achado clínico → hipótese → decisão → OSCE
```

# 1. Módulos

## 01_RESPIRATORIO

Foco:

- hipoxemia;
- dispneia;
- broncoespasmo;
- tosse e febre;
- dor pleurítica;
- exame físico respiratório;
- gasometria.

Competência central:

```text
diferenciar oxigenação, ventilação, perfusão e trabalho respiratório
```


# 3. Como estudar pelo MEDIX Learn

## Passo 1 — Leia a microaula

Objetivo: entender a ideia central.

Não tente decorar.  
Tente responder:

```text
qual é o mecanismo?
```

## Passo 2 — Ligue mecanismo com achado

Exemplo:

```text
broncoespasmo → via aérea estreita → sibilo → expiração prolongada → esforço
```

## Passo 3 — Procure gravidade

Pergunte sempre:

```text
o que me faria agir agora?
```

## Passo 4 — Resolva os mini-casos

Use os mini-casos para treinar decisão:

```text
qual hipótese?
qual mecanismo?
qual erro não posso cometer?
```

## Passo 5 — Faça as questões ativas

Não leia a resposta antes de tentar.

## Passo 6 — Use o mapa final

O mapa é revisão rápida antes do OSCE.

## Passo 7 — Vá para o MEDIX OSCE

Depois da trilha, treine atendimento:

```text
anamnese → exame físico → hipótese → conduta → feedback
```

```text

04_NEUROLOGIA
```

Mantenha os nomes dos arquivos exatamente como estão.

## Atenção

Se o app usa slugs específicos em `content/learn/`, o agente deve mapear:

```text
01_HIPOXEMIA.md → hipoxemia.md
02_DISPNEIA.md → dispneia.md
...
```

Não alterar parser sem necessidade.  
Não alterar visual sem pedido.  
Não criar gambiarras.

# 5. Critérios de qualidade

Uma trilha está boa quando você consegue responder:

- O que é?
- Por que acontece?
- Como aparece?
- Quando é grave?
- Qual hipótese principal?
- Qual diferencial perigoso?
- Qual exame muda conduta?
- Qual erro crítico evitar?
- Como apresentar no OSCE?

---

# 6. Limite de uso

Este pacote é educacional.

Ele não deve ser usado como:

- prescrição médica;
- protocolo institucional;
- orientação direta a paciente;
- substituto de professor/supervisor;
- decisão clínica real sem avaliação profissional.
