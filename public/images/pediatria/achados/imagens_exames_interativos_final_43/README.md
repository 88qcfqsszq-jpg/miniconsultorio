# Imagens finais - Exames Interativos Pediátricos

Pacote com 43 imagens organizadas nos diretórios finais do app.

Como usar:
1. Descompacte o ZIP.
2. Copie a pasta `public/` para a raiz do projeto.
3. Confirme substituição dos placeholders.
4. Rode o app e teste o painel de Exame Interativo.

Observação: a imagem de maus-tratos foi removida da lista final.
