# Casos OSCE v3 — modelo Asma Grave

Gerado com 61 casos.

Cada caso recebeu os campos do modelo aprovado no caso de Asma Grave:

- `sinaisVitais` com entrada e evolução por conduta;
- `exames` com laboratoriais inline e complementares originais;
- `feedbackDetalhado` com escala de 20 pontos, domínios, critérios e penalidades automáticas;
- preservação dos dados clínicos originais do banco anterior.

O caso 003 foi ajustado para `Asma Aguda Grave` e deve ser usado como referência estrutural.
