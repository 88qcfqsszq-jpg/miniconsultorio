# MEDIX Learn — Infectologia V3 Curado

Este pacote recria o sistema **Infectologia** do MEDIX Learn no novo padrão pedagógico:

- linguagem médica direta;
- foco em “você”, não “o aluno”;
- blocos curtos e profundos;
- fisiopatologia aplicada à decisão;
- semiologia que muda conduta;
- mini-casos reais;
- questões ativas específicas;
- mapa final útil;
- ponte para MEDIX OSCE Ciclo Básico e MEDIX OSCE Clínico.
