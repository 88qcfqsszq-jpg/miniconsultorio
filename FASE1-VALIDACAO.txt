✅ REGRA DE SCORING:
================================================================================

Fórmula:
  Score Total = (Termos Positivos Fortes × 50) +
                (Termos Positivos Secundários × 20) -
                (Termos Bloqueadores × 100) -
                (Termos Incompatíveis Graves × 200)

Decisão:
  if (Score Total >= min_score) {
    ACEITAR imagem
  } else {
    REJEITAR imagem
  }

Exemplos de Scores:

  PNEUMONIA - BOA:
  - "Left lower lobe consolidation. Pneumonia."
  - Termos: pneumonia (+50), consolidation (+50), lower lobe (+50)
  - Score: 150 >= 50 → ✅ ACEITA
  
  PNEUMONIA - MÁ:
  - "Normal chest radiograph. No acute findings."
  - Termos: "normal chest" (-100)
  - Score: -100 < 50 → ❌ REJEITA
  
  TB - BOA:
  - "Right upper lobe cavitary lesion. TB."
  - Termos: tuberculosis (+50), cavity (+50), upper lobe (+50)
  - Score: 150 >= 50 → ✅ ACEITA
  
  TB - MÁ:
  - "No acute cardiopulmonary abnormality."
  - Termos: "no acute" (-100)
  - Score: -100 < 50 → ❌ REJEITA
